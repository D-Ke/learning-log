## 二、代码实现步骤
**requests模块**
1. 发送数据：模拟浏览器对于url地址发送请求
2. 获取数据：获取服务器返回响应地址
3. 解析数据：提取我们需要的数据内容
4. 保存数据：把提取出来的数据保存到本地文件中

**drissionpage模块**
>自动化模块：模拟人的行为对浏览器进行相关操作

 1. 打开浏览器，访问网站
 2. 获取数据
 3. 解析数据
 4. 保存数据
**requests和drissionpage步骤第一步和第二步有所不同**
### 准备工作
>drissionpage官方文档：https://drissionpage.cn/get_start/installation/
### 代码实现
1. 打开浏览器，访问网站
```python
#导入自动化模块
from DrissionPage import ChromiumPage
#打开浏览器（实例化浏览器对象）
dp = ChromiumPage()  //dp是自定义变量名
#访问网站
dp.get('https://www.xiaohongshu.com/explore/68d3cb5c000000001302a3b2?xsec_token=ABxFZh2z_4gQ_qO0Qf7SxSVd5UfJ47VvjVfzHPyOtAdkk=&xsec_source=pc_search')
```
2. 获取数据
- 监听数据包:**监听数据包特征**
	- 当网站加载含有这个特征的数据包,可以直接获取数据响应
	- **特征:**`comment/page`
	- **细节:** 监听一定要在执行动作之前
3. 解析数据
- 字典取值:键值对取值,根据冒号左边的内容[键],提取冒号右边的内容[值]
- json字典数据,多层嵌套,需要一层一层提取
4. **保存数据**
- 保存数据到csv表格文件中 **固定写法**
### 最终代码
```python
# 导入自动化模块  
from operator import index  
  
from DrissionPage import ChromiumPage  
# 导入日期转换模块  
from datetime import datetime  
# 导入csv模块  
import csv  
# 创建文件对象  
f=open('data.csv',mode='w',encoding='utf-8',newline='')  
reader = csv.reader(f)  
# 字典写入方法  
csv_writer=csv.DictWriter(f,fieldnames=[  
  '昵称',  
  '日期',  
  '点赞',  
  '回复',  
  '地区',  
  '评论',  
])  
# 写入表头  
csv_writer.writeheader()  
# 打开浏览器  
dp=ChromiumPage()  
# 监听数据包  
dp.listen.start('comment/page')  
# 访问网站  
dp.get('https://www.xiaohongshu.com/explore/67517c580000000002039a98?xsec_token=ABHs4ZhIh5kFuK6FhnzyWeWCSLMhddB2GUNle-FUrYQHc=&xsec_source=pc_search')  
  
# 构建循环翻页  
for page in range(1,10):  
  print(f'正在采集第{page}页的数据内容')  
  # 等待数据包加载  
  r=dp.listen.wait()  
  # 获取响应数据->json字典数据  
  json_data=r.response.body  
    # 字典取值,提取评论数据所在列表  
  comments=json_data['data']['comments']  
  # for循环遍历,提取列表里的元素  
  for index in comments:  
  # 在循环中提取具体每条评论信息内容  
  # 获取index字典中所有的键  
  key_list=[i for i in index.keys()]  
  # 判断是否存在地区数据  
  if 'ip_location' in key_list:  
  ip_location=index['ip_location']  
  else:  
  ip_location='未知'  
  # 提取评论时间戳  
  t=str(index['create_time'])[:-3]  
  # 把时间戳转换成日期  
  date=str(datetime.fromtimestamp(int(t)))  
  dit={  
  '昵称':index['user_info']['nickname'],  
  '日期':date,  
  '点赞':index['like_count'],  
  '回复':index['sub_comment_count'],  
  '地区':ip_location,  
  '评论':index['content'],  
 }  csv_writer.writerow(dit)  
  print(dit)  
  id_=index['id']  
  # 定位标签  
  tab=dp.ele(f'#comment-{id_}')  
  # 下滑页面  
  dp.scroll.to_see(tab)
```

			)

		center=["40%", "50%"],
